## About

This repository is the new repository for my Final Integrated Project, my old repository just broke, in that I can
no longer do commits on it and I ended up having to create a new repository. Please use this repository going forward
the old one will be discarded once I transfer all my work from my old repository for the FIP over to this new one.
For the final integrated project, we are required to revive a defunct drink brand and create a refresh for it in the
modern day. We were given many different options, but I chose to revive the Quatro brand. The relaunch includes a
website as well as other multiple components. Some of these components were created in other assignments and later
used for the Final Integrated Project, such as a commercial, a cinemagraph for the website and drink bottles used as
"mockups" for the fictional revival of the brand.
